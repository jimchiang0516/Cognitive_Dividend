# 交流經驗分享(這一段內容需花3分鐘看完)
- 經過對方**意願**很重要！隱私，重要！
- **傾聽**最重要，不傾聽以理解對方需求就說話，跟自說自話無異，如同用自己的答案填別人的問題。
- **沉默是金**！少廢話(思考**講了之後的影響**)！以達到講話有份量、含金量。
  - **講出來的話與聽者有何關係、重不重要、緊不緊急？**
    - 思考是否**知道且有能力**滿足對方需求？
    - 換位思考！以免自說自話。
    - 己所不欲，勿施於人！
  - **信用需要時間**驗證，
    - **不隨便發誓、承諾**！
    - **想想為對方付出奉獻過什麼**？否則憑什麼交情去請求對方照自己意思？即便對方拒絕，也要欣然接受，因為付出本就是單方面的一廂情願！
  - 別人不問，不說。別人問，誠懇答覆。八卦，不說不傳不參與。
  - 思想可以真誠，但說話必須留三分話(因為要體諒對方是否能接受)。
- 意圖陳述明確 + 邏輯分明 + 言行一致
  - 說話的3個目的：
    - 溫暖人心，廣結善緣
    - 贏得認同，達成自己目標
    - 創造共同利益
  - 話多者(可能較有主見)，傾聽。話少者，主動傳接球。
  - 對理性的人，想辦法言之有理。對感性的人，想辦法能感動。
- 用詞要精確，避免產生情緒(話越長可能容易產生情緒)！所以言簡意賅！
- 學著克制即興，先明確主題，再用故事包裝。避免下指導棋、說教！
- 學習環境中最受歡迎之人的特質！藉由他們喜歡幫助誰、喜歡跟誰聊天，知道誰是最受歡迎的！
- 跟心眼好的人在一起交流，他不會做傷害你的事，假如意見不合，他只會默默遠離你。
- 君子之交，淡如水。
  - 要懂得拒絕人，每個朋友最多回適量訊息，除非狀況緊急，否則盡量要讓彼此有空間。
- 有趣幽默(反差意外沒想到的感覺、隱喻譬喻、猜謎腦筋急轉彎、冷笑話、說故事、雙關)
- 看見表情有異樣(面色愁容)的人 ⇨ 日安，你還好吧？ ⇨ 對方若發狂，就說不好意思，我認錯人了！
- 少點難怪...、不意外，多點**加油**！
