# 交流經驗、注意事項分享
- 經過對方**意願**很重要！！！隱私，重要！
- 沉默是金！！！！！花若盛開，蝴蝶自來。
- 學著克制即興，先明確主題，再用故事包裝。避免下指導棋、說教！
- 看見表情有異樣(面色愁容)的人 ⇨ 日安，你還好吧？ ⇨ 對方若發狂，就說不好意思，我認錯人了！
- 傾聽最重要，不傾聽對方需求就說話，跟自說自話無異，如同用自己的答案填別人的問題。
- 用詞要精確，避免產生情緒(話越長越容易產生情緒)！！！所以言簡意賅！
- 思想可以真誠，但說話必須留三分話(因為要體諒對方是否能接受)，信用需要時間驗證，不隨便發誓、承諾！！！
- 己所不欲，勿施於人！！！ 
- 換位思考！(不是每位都能接受)保持正道！
- 說話目的3個：
  - 溫暖人心，廣結善緣
  - 贏得認同，達成自己目標
  - 創造共同利益
- 找出專家學習他，找出環境中最受歡迎之人，學習他特質！藉由他們喜歡幫助誰、喜歡跟誰聊天，知道誰是最受歡迎的！
- 別人不問，不說。別人問，誠懇答覆。八卦，不說不傳不參與。
- 要懂得拒絕、無視人，每個朋友最多回適量訊息，除非狀況緊急，否則盡量要讓彼此有空間。
- 不遠不近的關係適合任何關係。
- 言簡意賅，言之有理，能感動人(需要感動，需要有情緒上的感覺)，不廢話。
- 意圖陳述明確 + 邏輯分明 + 言行一致
- 講話有份量、含金量。
- 有趣幽默(反差意外沒想到、隱喻譬喻、猜謎腦筋急轉彎、冷笑話、說故事、雙關)
- "知道且有能力"滿足對方需求
- 話多者(可能較有主見)，傾聽。話少者，主動傳接球。
- 跟心眼好的人再一起玩，心眼好本身與生俱來，心眼好是刻在骨子裡，他絕對不會做傷害你的事，不會捅刀子，只會默默遠離你。
- 少點難怪...、不意外，多點 加油！
